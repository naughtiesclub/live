  • Installed shadows rising game [pre-alpha]
  • Implemented new theme

Added new finctions:

  • New Gallery - TidyPics
  • Badges - The Badge plugin allows to award users with badges when they reach certain numbers of userpoints (not yet implemented). Alternatively, the badges can also be assigned manually.
  • Trusted users - This plugin is intended to be used as a dependency for other plugins that build functions specifically for trusted users. By default there are two methods users can be considered trusted:
  • 1. Administrator can set a user to be explicitly trusted
  • 2. Admin options allow for users to be trusted after a certain amount of membership time
  • Crushes - Maintains a list of 'Crushes' users have. If crushes are mutual, it triggers a notification.
  • Site Announcements - Inform site users about upcomming maintenance or just to tell them something important. These announcements will be shown on every page until the user has taken notice of them by clicking them away.

Pending

  • Block Users - This plugin gives a link on the user hover menu to block a user. Blocked users will not have access to pages owned by the blockee, and will not be able to send messages. Note that this will not necessarily block all content (if the content shows on a page not owned by the user), and will not block river (activity) entries.
  • River Privacy - By default, every change you make to your profile or posting you make, can be seen by the entire network. This will hopefully fix that, making your activity visible only to your friends. (Sets non-object oriented river items to private access (eg. friend relationships))
  • Feedback Option - Available from all pages.
  • TidyPics Plus - Improve/Add some TidyPics functionality. Extends the TidyPics plugin to include nicer galleries and a popup image viewer.
